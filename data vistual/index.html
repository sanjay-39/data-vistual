About DataVis Pro

An advanced data visualization system for the modern information age.

📊

Multiple Chart Types

Bar, Line, Pie, Donut, Scatter, and Radar charts powered by Chart.js 4.0 with smooth animations and gradient fills.

📁

Multi-Format Input

Upload CSV files (PapaParse), Excel spreadsheets (SheetJS), or plain text. All parsing happens in your browser.

✏️

Manual Entry

Build datasets from scratch with an interactive table editor — add/remove rows and columns, edit any cell inline.

🎮

Interactive

Hover tooltips, zoom and pan, real-time axis filtering, chart export as PNG, and multi-series support.

🌌

Anti-Gravity UI

Floating particle canvas, magnetic card hover effects, and smooth micro-animations for a premium experience.

🚀

Future: AI & AR/VR

Planned: AI-powered chart recommendations, augmented reality data overlays, and voice-driven data analysis.
